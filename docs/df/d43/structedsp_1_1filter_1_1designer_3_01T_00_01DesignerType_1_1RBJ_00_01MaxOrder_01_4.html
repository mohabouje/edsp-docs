Returns a filter which parameters represents the designed frequency response by using the RBJ formulas.

Parameters
sample_rateThe sampling frequency in Hz.
fcCenter Frequency or Corner Frequency in Hz, or shelf midpoint frequency, depending on which filter type.
gain_dbGain in dB, used only for peaking and shelving filters.
QDepending of the filter represents the Quality factor, the bandwidth in octaves (between -3 dB frequencies for BPF and notch or between midpoint (gain_db/2) gain frequencies for peaking EQ) or the "shelf slope" parameter.
Returns
Biquad filter with the computed coefficients.
