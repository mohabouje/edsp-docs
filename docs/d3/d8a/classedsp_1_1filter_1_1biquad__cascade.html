Detailed Description

template<typename T, std::size_t N>
class edsp::filter::biquad_cascade< T, N >

The BiquadCascade class implements a cascade of Biquad filters.

This class implements arbitrary order recursive (IIR) filters as a cascade of second order Biquad sections. In this implementation the output from the first filter is the input to the second, and so on.

Template Parameters
TValue type
NNumber representing the maximum size (number of Biquad).
