Detailed Description

template<typename T, typename Allocator = std::allocator<T>>
class edsp::filter::moving_median< T, Allocator >

This class implement a cumulative moving median (rolling median or running median) filter.

Given a series of numbers and a fixed subset size, the first element of the moving median is obtained by taking the median of the initial fixed subset of the number series. Then the subset is modified by "shifting forward"; that is, excluding the first number of the series and including the next value in the subset.
